Crypto Data Feeds

August Data Feeds provide data that is aggregated from many data sources by a decentralized set of independent node operators. The Decentralized Data Model which is not fully implemented is modeled after chainlink.

However, there are some exceptions where data for a feed can come only from a single data source or where data values are calculated. In such a situation, August Oracle distributes the provision of this data to different nodes and coordinates their coperation.

Go to Data Feed fo find out more on how to integrate the service.

SMTP

Due to the limitations of consensus driven open blockchains, salient services like SMTP have remained a challenge to support.

But to enable full blown web applications to be run onchain on the Internet Computer, it is necessary to provision a means for services like SMTP to equip canisters with the ability to send emails to their users.

Go to SMTP Page to integrate our SMTP service and start sending emails from your canisters.

Canister Snapshots

Coordinated replication of data is what gives blockchains the power to host applications in a fully decentralized way and it didn't come without its troubles.

Snapshots brings the power of tradition backups to canisters. while taking an application might seem anti decentralization, some applications might need the ability to keep snapshots of their canisters making rollbacks and analytics possible and easy.

Go to Snapshots for more info on this service.
